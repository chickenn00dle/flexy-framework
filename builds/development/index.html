Flexy

A FlexBox based framework.

Welcome to Flexy, a very simple flexbox based framework designed and developed by me as a learning experience and a tool to use for all of my future web projects.

Flexy is made entirely of css and is made to be as minimalistic as possible. This way you are free to build upon it however you see fit.
