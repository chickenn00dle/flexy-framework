Flexy

A FlexBox Framework

Flexy Guide

Welcome to Flexy, a very simple flexbox based framework designed and developed by me as a learning experience and a tool to use for all of my future web projects.

Flexy is a work in progress, made primarily with css and is super minimal so it can be built upon however you see fit. If you'd like to use Flexy as a starting point for your web projects, you can get started by cloning the GitHub Repository. If you'd like to support Flexy, give me a shoutout on Twitter , or hit the donate button below. Thanks in advance!
