Framework

A FlexBox based framework.

The "grid" system, or how the content is laid out, is essential in any framework. While the sytsem itself is not very different from how many others work, the mentality behind the naming conventions differs ever so slightly.

There isn't too much explaining to do, so let's get right to it!

The Shell

All page content is placed into a Flexy Shell. As the name implies, the shell serves as a sort of empty "shell" to place bits of content as you see fit. There need only be one shell per page and for semantics sake, class shell should be applied to the pages main tag.

The Unit is where you will place all of your content. All headings, paragraphs, anchor tags, and etc. belong in here.
