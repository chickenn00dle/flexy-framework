Grid

The Flexy System

The Flexy grid was designed to make life simple. First, we establish a flexy shell through which other flexible units can be placed.

All page content is placed into the Flexy Shell. As the name implies, it serves as an empty "shell" within which you will place all modules and units. You should only need one shell per page and for semantics sake, shells should be applied to your pages main tag.

Modules house groups of similar units. A module serves as the second layer of structure after a pages shell. Place your module divs within section tags.

Units are where we place actual content. All headings, paragraphs, anchor tags, and etc.

<main class="shell">
    <section>
        <div class="module">
            <div class="unit">
                // Your Code Here
            </div>
        </div>
    </section>
</main>

Units

By default, units will take up even amounts of space across the width of the viewport. All units placed within a module will grow evenly as the page width increases and will shrink evenly as the page width decreases.

If you'd like to specify the width of a unit, simply add 1-10 to the unit in question. (Units within a single module should total to less than 10.) Adjacent units may also be left unspecified to simply take up the remaining space.

<div class="unit-8">
    <span>Unit 1</span>
</div>
<div class="unit">
    <span>Unit 2</span>
</div>

Or if you'd like to center or right-justify a unit, simply add module-center / module-right to the parent module:

        Unit
    </div>
</div>
