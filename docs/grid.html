Grid

The System

The Flexy grid was designed to make life simple. First, we establish a flexy shell through which other flexible units can be placed.

The Shell

All page content is placed into a Flexy Shell. As the name implies, it serves as an empty "shell" within which you will place all modules and units. You should only need one shell per page and for semantics sake, shells should be applied to your pages main tag.

The Module

Modules house groups of similar units. A module serves as the second layer of structure after a pages shell. Place your module divs within section tags.

The Unit

Units are where we place actual content. All headings, paragraphs, anchor tags, and etc.
